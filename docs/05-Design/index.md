# 05 Design

## Design Principles
The visual theme is **"Energetic VIP Party"**—luxurious yet warm, modern yet inviting. It exclusively uses a **Dark Mode** theme.

## Design System
- **Backgrounds:** Deep darks (`#0a0a0a`) with dynamic gradients (`#1a0a2e` to `#0f0f0f`).
- **Accents:** Neon Violet/Indigo gradients (`#6366f1` to `#a855f7`) used sparingly for CTAs, decorative orbs, and borders.
- **Typography:**
  - **Headings:** `Rubik` (Bold/Heavy, for impact).
  - **Body:** `Noto Sans Hebrew` (Regular, for readability).
- **Layout:** Right-to-Left (RTL) first. Utilizes Bento Grids, Full-bleed Hero sections, and Sticky navigation.
- **Cards:** Straight corners (`rounded-none`), top accent borders, semi-transparent backgrounds (`bg-white/5`).

## Photography Style
**"Exclusive Paparazzi"** — Direct flash, high contrast, capturing natural moments in motion against dark backgrounds.

## Motion & Animation
- **Scroll Reveals:** 400-600ms ease-out animations as sections enter the viewport.
- **Staggered Bento:** Cards load sequentially (80-120ms delay).
- **Micro-interactions:** 150-220ms hover states on buttons and cards (translateY + shadow).
