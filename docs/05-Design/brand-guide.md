# חלק א׳ — זהות המותג

## מהות

מרים סגל היא יוצרת תוכן בתחום הלייפסטייל וחיי המשפחה. האתר משמש כהיברידית: **חגיגת יום הולדת דיגיטלית (B2C)** ו-**כלי מכירות מסחרי למותגים (B2B)**.

## DNA וערכים

| ערך | ביטוי ויזואלי |
|-----|----------------|
| **אנרגטית** | אקסנט בוהק, תנועה, אנימציות עדינות |
| **חברותית** | פלאש עריכתי, רגעים טבעיים, UGC |
| **אלגנטית** | Dark Mode, טיפוגרפיה נקייה, פינות ישרות |

## קונספט ויזואלי

**"מסיבת VIP אנרגטית"** — יוקרתי אך חם, מודרני אך מזמין.

# חלק ד׳ — כרטיסים וסקשנים

## מסגרת

- **סגנון:** קו עליון אקסנט עבה (`border-top: 2px`)
- **צבע:** `rgba(139,92,246,0.6)` או גרדיאנט כחול→סגול
- **פינות:** ישרות (`rounded-none`) — אין עיגול

## סוגי כרטיסים

| סוג | סגנון |
|-----|--------|
| רגיל | `border-t-2 border-violet-500/60`, `bg-white/5` |
| הדגשה | קו גרדיאנט עליון 3px (`linear-gradient`) |
| קו צד | קו אנכי בשמאל (מילון, הצבעות) |

## Hover

- `translateY(-4px)` + shadow מוגבר
- משך: 180ms ease-out

# חלק ה׳ — תמונות ועיבוד צילום

## בריף צילום

**סגנון:** פפראצי אקסקלוסיבי — תאורת פלאש ישירה (Editorial Flash), קונטרסט גבוה, לכידת רגעים טבעיים בתנועה על רקע כהה.

## הנחיות שימוש בתמונות

### טקסטורה ומצב

| קריטריון | דרישה |
|----------|--------|
| **איכות** | מינימום 1200px ברוחב (2x לרטינה) |
| **פורמט** | WebP מועדף; JPG fallback |
| **נפח** | עד 200KB לאופטימיזציה |

### עיבוד ועריכה

| אלמנט | הנחיה |
|-------|--------|
| **קונטרסט** | גבוה — shadows עמוקים, highlights בולטים |
| **טמפרטורה** | ניטרלית עד מעט חמה (לא קר) |
| **רוויה** | בינונית — לא oversaturated |
| **Vignette** | עדין אופציונלי — דגש למרכז |
| **רקע** | להעדיף תמונות עם רקע כהה או לאבזר overlay כהה |

### פריסה ומבנה

| סוג אזור | יחס גובה–רוחב | מיקום |
|----------|----------------|--------|
| **Hero** | 16:9 או 3:2, full-bleed | מילוי viewport, `object-fit: cover` |
| **Case Study** | 4:3 או 1:1 | בתוך כרטיס, עם קו עליון |
| **גלריה פפראצי** | 1:1 או 4:5 | grid, `object-fit: cover` |
| **פרופיל / אודות** | 3:4 portrait | ממוקד, עם מסגרת קו עליון |
| **תמונות UGC** | גמיש — לשמור על aspect ratio | grid עם ג׳אסטיפיקציה |

### Overlay ורקע

- תמונות Hero: overlay כהה (`rgba(0,0,0,0.4)`–`0.6`) על מנת לשמור קריאות טקסט.
- טקסט על תמונה: רק על אזורים כהה enough — אחרת להוסיף gradient overlay.
- תמונות כרטיסים: רקע `#0f0f0f` או `bg-[#0f0f0f]` תחת התמונה.

### אזור יום הולדת vs אזור כללי

| אזור | טון צילום |
|------|------------|
| **כללי / Media Kit** | מקצועי, אלגנטי — פחות "מסיבה", יותר premium |
| **מתחם יום הולדת** | אנרגטי, רגעי — פלאש, תנועה, תחפושות, קהל |

# חלק ז׳ — Motion ואנימציות

| סוג | משך | Easing | שימוש |
|-----|-----|--------|--------|
| Hover | 150–220ms | ease-out | כפתורים, כרטיסים |
| Scroll reveal | 400–600ms | ease-out | סקשנים |
| Stagger | 80–120ms | ease-out | כרטיסי Bento |
| Transition דף | 300ms | ease-in-out | מעבר בין דפים |

**כללי גלילה:** סקשן נחשף ב־70–80% viewport. Scroll-driven (GSAP או Intersection Observer).
