# 08 Feedback

## Recent Iterations (Feb 24, 2026)
- **Metric Removal:** The "אחוז מעורבות" (Engagement Percentage) metric was deemed noisy and inaccurate for non-TikTok platforms. It was removed from the Admin panel and the Homepage stats grid.
- **UI Fixes:** Horizontal scrolling issues caused by the Brands Carousel marquee were fixed by enforcing `overflow-x-hidden` on root layouts.
- **Admin Enhancements:** Feedback indicated the need for inline editing. Admin panels for Brands and Case Studies were updated from "Add/Delete only" to full CRUD with inline edit forms.

## Outstanding Change Requests
- **Data Entry:** Replace all `placeholder` data with actual media kit statistics and brand assets.
- **Asset Management:** Implement Firebase Storage uploads for Case Study images instead of relying on external URLs.
- **Sorting:** Add drag-and-drop reordering functionality for Admin lists (Brands, Case Studies).
