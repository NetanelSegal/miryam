# 02 Frameworks

## Technology Stack
- **Frontend:** React 19.1.0
- **Routing:** React Router 7.13.0
- **Styling:** Tailwind CSS 4.0.0
- **Build Tool:** Vite 7.1.0
- **Language:** TypeScript (~5.7.0)

## Backend & Infrastructure
- **Platform:** Firebase (SDK ^11.7.1)
- **Database:** Cloud Firestore (Primary CRUD), Realtime Database (for live votes)
- **Authentication:** Firebase Auth (Google Sign-in, Admin Whitelisting)
- **Storage:** Firebase Storage (for image uploads)
- **Hosting:** Vercel (indicated by `vercel.json`) / Firebase Hosting

## Key Libraries
- **Animations:** `motion` (Framer Motion ^12.34.3)
- **Icons:** `lucide-react` (^0.474.0)
- **Utilities:** `html2canvas`, `qrcode`

## Development Tools
- **Linting:** ESLint 9 (with `@eslint/js`, `typescript-eslint`)
- **Type Checking:** `tsc`
