## Access Rules

- **Discovery & Requirements:** Check `01` and `03` when starting new modules.
- **Architecture & Dev:** Check `04` and `06` before writing backend/logic code.
- **Design:** Check `05` for all frontend modifications.
- **Testing:** Always update/reference `07` during the validation phase.
