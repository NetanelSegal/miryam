# 09 Analytics

## Key Performance Indicators (KPIs)
While the site acts as a Media Kit presenting KPIs to brands, we also need to track the performance of the site itself:

- **Party Engagement:** Number of unique participants, Trivia completion rates, total Votes cast, and Blessings submitted.
- **Media Kit Conversion:** Clicks on the Contact form, downloads of the Media Kit (if applicable), and time spent on the Case Studies section.
- **Live Event Uptime:** Stability of the `/live` rotating display during the physical event.
