# 01 Discovery

## Project Overview
The **Miryam Segal Website** is a hybrid web application designed to serve dual purposes: it acts as a digital, interactive hub for a birthday party celebration, while also functioning as a professional Media Kit for B2B brand collaborations. Miryam Segal is a content creator focusing on lifestyle and family.

## Core Vision
To create a "VIP Party" digital experience that is energetic and celebratory for fans and friends, while maintaining an elegant, high-contrast, professional presentation for brands looking to sponsor or collaborate.

## Target Audience
1.  **B2C (Fans & Party Guests):** Followers participating in interactive activities like Trivia, Voting, reading the Dictionary, and leaving Blessings.
2.  **B2B (Brands & Agencies):** Potential sponsors viewing the Media Kit, case studies, audience statistics, and past brand collaborations.

## Project Phases
- **Phase 1-4:** Core infrastructure, Homepage (Media Kit), Party Zone shell, UI component library, Animation system, and Admin Panel setup (Completed).
- **Phase 5:** Firebase Integration for real-time features (Authentication, Firestore, Storage) (In Progress).
- **Phase 6:** Launch, QA, and replacing placeholder statistics with real data.
